# 🚀 Cryptocurrency Price Tracker  

A simple and interactive web application to track real-time cryptocurrency prices using CoinStats API.  

---

## 📌 Project Description  

This project is a Cryptocurrency Price Tracker that allows users to:  
- View top 10 cryptocurrencies  
- Search for a specific coin  
- See real-time price updates  
- View coin name, price, and icon  

The application fetches live data using an API and displays it dynamically on the webpage.  

---

## 🛠️ Technologies Used  

- HTML  
- CSS  
- JavaScript  
- CoinStats API  

---

## ⚙️ Features  

- 🔍 Search any cryptocurrency (e.g., Bitcoin, Ethereum)  
- 📊 Display top 10 cryptocurrencies  
- 💰 Real-time price updates  
- 🖼️ Coin icons display  
- ⚠️ Error handling (invalid coin / API error)  
- ⏳ Loading indicator
